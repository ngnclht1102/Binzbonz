# Task 2 — QA: Monorepo Setup

## Type: QA (Manual)

## Verify
1. **Clean install** — delete `node_modules` everywhere, run `pnpm install` from root → no errors
2. **Dev servers** — run `pnpm dev` from root:
   - [ ] NestJS API logs "Listening on port 3001" (or similar)
   - [ ] `curl http://localhost:3001` returns a response (even 404 is fine — server is alive)
   - [ ] Next.js loads at `http://localhost:3000` in browser — see default page
   - [ ] agent-runner starts without crash (can just log "agent-runner started")
3. **Build** — run `pnpm build` from root:
   - [ ] All three apps compile with exit code 0
   - [ ] `apps/api/dist/` exists
   - [ ] `apps/web/.next/` exists
   - [ ] `agent-runner/dist/` exists
4. **Turbo** — run `pnpm build` a second time → turbo cache hit (near-instant)

## Pass Criteria
All checks above pass. If any fail, reopen Task 1.
