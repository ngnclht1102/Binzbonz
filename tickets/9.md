# Task 9 — Hierarchy CRUD Module (MVP, Sprint, Epic, Feature)

## Type: Backend

## Description
Build the Hierarchy NestJS module with CRUD endpoints for MVPs, Sprints, Epics, and Features. These are organisational nodes — no agent work happens here.

## Endpoints

### MVPs
| Method | Path | Description |
|--------|------|-------------|
| GET | `/projects/:projectId/mvps` | List MVPs for a project |
| POST | `/projects/:projectId/mvps` | Create MVP |
| PATCH | `/mvps/:id` | Update MVP |
| DELETE | `/mvps/:id` | Delete MVP (cascades) |

### Sprints
| Method | Path | Description |
|--------|------|-------------|
| GET | `/mvps/:mvpId/sprints` | List sprints for an MVP |
| POST | `/mvps/:mvpId/sprints` | Create sprint |
| PATCH | `/sprints/:id` | Update sprint |
| DELETE | `/sprints/:id` | Delete sprint (cascades) |

### Epics
| Method | Path | Description |
|--------|------|-------------|
| GET | `/sprints/:sprintId/epics` | List epics for a sprint |
| POST | `/sprints/:sprintId/epics` | Create epic |
| PATCH | `/epics/:id` | Update epic |
| DELETE | `/epics/:id` | Delete epic (cascades) |

### Features
| Method | Path | Description |
|--------|------|-------------|
| GET | `/epics/:epicId/features` | List features for an epic |
| POST | `/epics/:epicId/features` | Create feature |
| PATCH | `/features/:id` | Update feature |
| DELETE | `/features/:id` | Delete feature (cascades) |

## Files
- `apps/api/src/hierarchy/hierarchy.module.ts`
- `apps/api/src/hierarchy/mvps.controller.ts` + `mvps.service.ts`
- `apps/api/src/hierarchy/sprints.controller.ts` + `sprints.service.ts`
- `apps/api/src/hierarchy/epics.controller.ts` + `epics.service.ts`
- `apps/api/src/hierarchy/features.controller.ts` + `features.service.ts`
- DTOs for each entity (create + update)

## Notes
- Validate parent exists on create (e.g. project must exist to create MVP)
- Cascade deletes: deleting an MVP deletes its sprints → epics → features
- Each entity returns its children count in list responses

## Acceptance Criteria
- [ ] Full CRUD works for MVP, Sprint, Epic, Feature
- [ ] Parent-child relationships enforced (can't create sprint without valid MVP)
- [ ] Cascade delete works top-down
- [ ] 404 returned for missing parent or entity
