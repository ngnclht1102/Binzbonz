# Task 6 — QA: Database, Schema & Seed

## Type: QA (Manual)

## Verify

1. **Fresh start** — delete `apps/api/data/postgres/` if it exists, then run `pnpm dev --filter api`
   - [ ] Logs show embedded Postgres initialising (first run)
   - [ ] Logs show TypeORM synchronising schema
   - [ ] Logs show seed data being inserted (8 actors)
   - [ ] API is listening on port 3001

2. **Verify tables** — connect to the embedded Postgres:
   ```bash
   psql postgres://binzbonz:binzbonz@127.0.0.1:54329/binzbonz
   ```
   - [ ] `\dt` shows all tables: actor, project, mvp, sprint, epic, feature, task, comment, project_comment, memory_file, wake_event
   - [ ] `SELECT * FROM actor;` returns 8 rows (1 ctbaceo, 6 devs, 1 brian)
   - [ ] `\d task` shows foreign keys to feature, actor (assigned_agent, created_by), and self-ref (parent_task)

3. **Restart** — stop and restart `pnpm dev --filter api`
   - [ ] Logs show reusing existing Postgres cluster (no re-init)
   - [ ] Seed says "already exists" or inserts 0 new rows
   - [ ] All data still present

4. **External mode** — set `DATABASE_URL` to a running external Postgres, restart API
   - [ ] Embedded Postgres is NOT started
   - [ ] TypeORM connects to external DB instead

5. **Clean shutdown** — press `Ctrl+C`
   - [ ] No orphan `postgres` process left running (`ps aux | grep postgres`)

## Pass Criteria
All checks above pass. If any fail, reopen Task 3, 4, or 5 as appropriate.
